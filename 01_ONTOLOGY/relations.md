# Ontologie minimale — Relations

Ce document décrit les relations opératoires entre les objets.
Aucun cas particulier n’est introduit ici.

---

## Projection
**I → C**  
L’intention est projetée dans un espace contraint.

## Transformation
**(I → C) —O→ A**  
L’opérateur transforme l’intention projetée sous contraintes
en un artefact.

## Évaluation
**A → T**  
L’artefact est évalué selon sa transmissibilité.

---

## Chaîne canonique
**I → C → O → A → T**
